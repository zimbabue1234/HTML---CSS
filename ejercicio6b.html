Lenguaje de marcas y sistemas de gestión de información

El módulo de Lenguajes de marcas es un módulo de 1º con una carga horaria de 3 horas semanales impartidas a lo largo de 3 trimestres.

Los contenidos del módulo son los siguientes:

  1. Características de los lenguajes de marcas
  2. Lenguajes para el almacenamiento y transmisión de información
  3. El lenguaje XML
  4. XPath y XQuery
  5. El lenguaje JSON
  6. El lenguaje Marktown
  7. El lenguaje HTML 5
  8. El lenguaje CSS3
  9. El lenguaje de programación Javascript
